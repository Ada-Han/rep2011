## 项目概述

年龄结构种群模型是种群动力学中的重要模型，用于描述种群密度随空间、年龄和时间的变化。本项目通过数值方法求解该模型，并验证不同数值方法的收敛性。

### 模型方程

年龄结构种群模型的基本方程为：

```
∂u/∂t + ∂u/∂a = κ∇²u - μ(a)u
```

其中：
- `u(x, y, a, t)` 是种群密度
- `κ` 是扩散系数
- `μ(a)` 是死亡率函数
- `∇²` 是拉普拉斯算子

### 边界条件

- 出生边界条件：`u(x, y, 0, t) = β∫₀^A u(x, y, a, t) da`
- Neumann边界条件：`∂u/∂n = 0`（在空间边界上）

## 实验设计

### 第一组实验（Group 1）

**目标**：验证时间步长收敛性

**参数设置**：
- 扩散系数：`κ = 0.01`
- 出生率系数：`β = 1.3025`
- 空间区域：`[0, 0.2] × [0, 0.2]`
- 最大年龄：`A_MAX = 15.0`
- 终止时间：`T_FINAL = 12.0`
- 评估时间：`T_EVAL = 3.0`
- 空间网格步长：`h = 0.005`
- 时间步长列表：`dt = 1/10, 1/20, 1/30, 1/40, 1/60, 1/80`

**死亡率函数**：
1. `μ(a) = 2`（常数死亡率）
2. `μ(a) = 2.457 + 2a(a - 2)`（年龄相关死亡率）

**收敛阶**：预期达到二阶时间收敛

### 第二组实验（Group 2）

**目标**：验证空间网格收敛性

**参数设置**：
- 扩散系数：`κ = 0.01`
- 出生率系数：`β = 1.3025`
- 空间区域：`[0, 1] × [0, 1]`
- 最大年龄：`A_MAX = 15.0`
- 终止时间：`T_FINAL = 12.0`
- 评估时间：`T_EVAL = 3.0`
- 空间扰动幅度：`δ = 0.20`
- 固定时间步长：`dt = 1/240`
- 空间网格列表：`nx = 8, 16, 32, 64, 128`
- 参考空间网格：`nx = 512`

**初值形式**：`u(x, y, a, 0) = u₀(a) × (1 + 0.2ψ_h(x, y))`

**收敛阶**：预期达到二阶空间收敛

### 第三组实验（Group 3）

**目标**：使用制造解验证空间和时间收敛性

**参数设置**：
- 扩散系数：`κ = 0.01`
- 死亡率常数：`μ = 2.0`
- 空间区域：`[0, 1] × [0, 1]`
- 最大年龄：`A_MAX = 1.0`
- 终止时间：`T_FINAL = 1.0`
- 评估时间：`T_EVAL = 1.0`

**制造解**：`u(x, y, a, t) = cos(πx)cos(πy)exp(-(a + t))`

**源项**：`f = (-2 + 2π²κ + μ)u`

**收敛测试**：
1. 空间收敛：固定时间步长，变化空间网格
2. 时间收敛：固定空间网格，变化时间步长

### 第四组实验（Group 4）

**目标**：空间-时间耦合加密收敛性测试

**参数设置**：
- 继承第三组的基本参数
- 耦合空间网格列表：`nx = 8, 16, 32, 64, 128`
- 时间步长函数：`dt = da = h = 1/nx`

## 数值方法

### 有限元方法（FEM）

- 使用P1三角形元
- 标准Galerkin方法
- 3点高斯积分
- 向后Euler时间离散

### 虚拟元方法（VEM）

- 使用一阶虚拟元
- 多边形网格支持
- 投影算子计算
- 向后Euler时间离散

## 结果验证

实验结果与论文数据进行对比，验证数值方法的正确性。误差计算使用相对L2范数：

```
error = ‖u_num - u_ref‖₂ / ‖u_ref‖₂
```

其中：
- `u_num` 是数值解
- `u_ref` 是参考解（细网格解或制造解）

## 注意事项

1. **网格要求**：空间网格步长必须能整除区域尺寸
2. **时间步长**：`dt` 必须能整除年龄区间和评估时间
3. **计算资源**：细网格实验可能需要较长时间运行
4. **结果保存**：实验结果会保存到相应的输出文件中
