# REP2011 二维 FEM 基线测试

本文档汇总 `group1_2d.py` 到 `group4_2d.py` 的 FEM 基线参数设置和已有实验结果，后续测试 VEM 时可按相同参数进行对照。

通用约定：

- 年龄步长和时间步长绑定：`da = dt`。
- 当前求解器要求 `dt` 能整除年龄区间和评估时间。
- 表中误差均为指定评估时刻的相对误差。

## VEM 对比检查清单

用 VEM 复现实验时保持以下设置一致：

- 使用相同的空间区域、扩散系数 `kappa`、死亡率、年龄区间、评估时间和 `dt = da` 序列。
- 保持相同的相对误差定义：第一组和第二组为空间-年龄 L2 相对误差，第三组和第四组为空间-年龄 L2 相对误差以及 H1 半范数相对误差。
- 第二组使用相同参考设置：`NX_REFERENCE = 512`，`DT_SPACE = 1/240`。
- 第三组和第四组使用相同制造解和源项。
- 除非 VEM 的时间-年龄求解器明确支持非均匀最后一步，否则保留 `dt` 对年龄区间和评估时间的整除条件。
